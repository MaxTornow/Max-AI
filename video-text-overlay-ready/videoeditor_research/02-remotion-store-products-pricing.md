# Remotion Store Catalog

## Overview

Remotion offers purchasable products through their store that provide pre-built functionality, significantly reducing development time. These are production-ready components that can be integrated into custom projects.

---

## Premium Products

### Editor Starter Kit - $600

**What You Get:**
- Complete video editor UI built with Remotion
- Timeline component with tracks
- Layer management system
- Drag-and-drop asset handling
- Real-time preview integration
- Export functionality
- React-based, fully customizable

**Why It Matters:**
- Saves 2-4 weeks of development time
- Battle-tested UI patterns
- Includes complex state management
- Source code included for customization

**Use Case:** Building a full video editor like Submagic

---

### Timeline Component - $300

**What You Get:**
- Professional timeline UI component
- Multiple track support
- Zoom in/out functionality
- Playhead control
- Time ruler with markers
- Clip trimming handles
- Drag-and-drop reordering

**Why It Matters:**
- Timeline is the hardest UI to build from scratch
- Pixel-perfect scrubbing
- Performance optimized for many clips

**Use Case:** Adding timeline editing to existing Remotion project

---

### Animated Captions - $100

**What You Get:**
- Word-by-word caption animations
- 10+ animation presets (bounce, typewriter, highlight, etc.)
- Customizable styling system
- Whisper integration ready
- Mobile-friendly sizing

**Why It Matters:**
- Replicates Submagic's core feature
- Proven animations that look professional
- Ready to use with AI transcription

**Use Case:** Auto-captions feature like TikTok/Submagic
